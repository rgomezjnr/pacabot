# Alpaca Trading Reference Repositories

Collection of Alpaca-related repositories such as programming client libraries, integrations with other trading software/platforms, and trading bots.

[alpacahq/cli: CLI for Trading API](https://github.com/alpacahq/cli)

[alpacahq/Alpaca-API: The Alpaca API is a developer interface for trading operations and market data reception through the Alpaca platform.](https://github.com/alpacahq/Alpaca-API)

[alpacahq/alpaca-py: The Official Python SDK for Alpaca API](https://github.com/alpacahq/alpaca-py)

[alpacahq/alpaca-trade-api-python: Python client for Alpaca's trade API](https://github.com/alpacahq/alpaca-trade-api-python)

[alpacahq/alpaca-trade-api-js: Node.js library for Alpaca Trade API.](https://github.com/alpacahq/alpaca-trade-api-js)

[alpacahq/alpaca-mcp-server: Alpaca’s official MCP Server lets you trade stocks, ETFs, crypto, and options, run data analysis, and build strategies in plain English directly from your favorite LLM tools and IDEs](https://github.com/alpacahq/alpaca-mcp-server)

[alpacahq/broker-fastapi-backend: A reference backend for Alpaca's Broker API utilizing Python & FastAPI](https://github.com/alpacahq/broker-fastapi-backend)

[alpacahq/zipline: Zipline-Live, a Pythonic Algorithmic Trading Library](https://github.com/alpacahq/zipline)

[alpacahq/notebooks: A collection of Jupyter notebooks for getting started with Alpaca](https://github.com/alpacahq/notebooks)

[alpacahq/awesome-quant: A curated list of insanely awesome libraries, packages and resources for Quants (Quantitative Finance)](https://github.com/alpacahq/awesome-quant)

[alpacahq/supertrend_crypto_bot: An Alpaca Live Crypto Trading Bot built with Python!](https://github.com/alpacahq/supertrend_crypto_bot)

[alpacahq/Alpaca-Flutter-Demo-App: A boilerplate trading application to trade crypto on Alpaca. Made with Flutter and Dart.](https://github.com/alpacahq/Alpaca-Flutter-Demo-App)

[alpacahq/sp100algo: Follow S&P100](https://github.com/alpacahq/sp100algo)

[alpacahq/alpaca-backtrader-api: Alpaca Trading API integrated with backtrader](https://github.com/alpacahq/alpaca-backtrader-api)

[alpacahq/vectorbt: Find your trading edge, using the most powerful toolkit for backtesting, algorithmic trading, and research.](https://github.com/alpacahq/vectorbt)

[shirosaidev/stockbot: Alpaca algo stock trading bot](https://github.com/shirosaidev/stockbot)
